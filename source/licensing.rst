Scratch
-------

Pytch does not use any Scratch code, but we have drawn heavily from it
for inspiration and guidance.

* https://scratch.mit.edu/


Skulpt
------

The Python implementation used by Pytch is Skulpt.

* `Skulpt homepage <https://skulpt.org/>`_
* :doc:`Skulpt licence <licences/skulpt>`



Web app
-------

The Pytch web app is built using `React <https://reactjs.org/>`_ and
many other projects.  In turn, many of these projects rely on other
projects.  Please see each individual project's homepage for further
information.  Their licences are as follows.

The React framework
^^^^^^^^^^^^^^^^^^^

The JavaScript front-end is written using React.

* `React homepage <https://reactjs.org/>`_
* :doc:`React licence <licences/react>`

The Redux framework
^^^^^^^^^^^^^^^^^^^

To ease state management, Pytch uses Redux and Easy-Peasy.

* `Redux homepage <https://redux.js.org/>`_
* :doc:`Redux licence <licences/redux>`
* `Easy-Peasy homepage <https://easy-peasy.now.sh/>`_
* :doc:`Easy-Peasy licence <licences/easy-peasy>`

The Reach Router framework
^^^^^^^^^^^^^^^^^^^^^^^^^^

To implement navigation between URLs within the webapp, Pytch uses
Reach Router.

* `Reach Router homepage <https://reach.tech/router/>`_
* :doc:`Reach Router licence <licences/reach-router>`

The Bootstrap framework
^^^^^^^^^^^^^^^^^^^^^^^

For presentation and UI components, Pytch uses Bootstrap and React
Bootstrap.

* `Bootstrap homepage <https://getbootstrap.com/>`_
* :doc:`Bootstrap licence <licences/bootstrap>`
* `React Bootstrap homepage <https://react-bootstrap.github.io/>`_
* :doc:`React Bootstrap licence <licences/react-bootstrap>`

The Dexie database framework
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

To store your projects, Pytch uses the Dexie wrapper around the
underlying IndexedDB.

* `Dexir homepage <https://dexie.org/>`_
* :doc:`Dexie licence <licences/dexie>`

The Ace code editor
^^^^^^^^^^^^^^^^^^^

The code editor pane in Pytch is `Ace <https://ace.c9.io/>`_.

* `Ace homepage <https://ace.c9.io/>`_
* :doc:`Ace licence <licences/ace-editor>`

Font Awesome icons
^^^^^^^^^^^^^^^^^^

Pytch uses various icons provided by Font Awesome.

* `Font Awesome homepage <https://fontawesome.com/>`_
* :doc:`Font Awesome licence summary <licences/font-awesome>`

File Saver and JSZip
^^^^^^^^^^^^^^^^^^^^

To allow 'download as zip', Pytch uses File Saver and JSZip.

* `File Saver Homepage <https://github.com/eligrey/FileSaver.js>`_
* :doc:`File Saver licence <licences/file-saver>`
* `JSZip Homepage <https://stuk.github.io/jszip/>`_
* :doc:`JSZip licence <licences/jszip>`
